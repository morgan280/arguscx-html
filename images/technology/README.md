# Technology page images

Add your own images here and update `client/src/pages/technology.tsx` to use them.

Suggested filenames (replace the placeholder URLs in the `PLACEHOLDER_IMAGES` object):

- **hero.jpg** – Hero section (wide; e.g. 1440×560)
- **data-insights.jpg** – Data & Insights section (e.g. 800×500)
- **cx-transformation.jpg** – CX Transformation section (e.g. 800×500)
- **cx-platform.jpg** – CX Platform section (e.g. 800×500)

Then in `technology.tsx`, set for example:

```ts
hero: "/images/technology/hero.jpg",
```
